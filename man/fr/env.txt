ENV
NOM
env - affiche les variables d'environnement
SYNOPSIS
env
DESCRIPTION
Affiche les variables d'environnement courantes.
FONCTIONS AUTORISÉES

write()
malloc() / free()

VALEUR DE RETOUR

Renvoie 0 en cas de succès
Renvoie 1 si l'écriture échoue

NOTES

Affiche chaque variable sur une nouvelle ligne au format : nom=valeur
Aucun tri n'est requis
Aucun argument ou option n'a besoin d'être supporté
